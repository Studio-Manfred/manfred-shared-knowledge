# manfred-a11y

Accessibility skills for design, development, and pre-release QA against WCAG.

## Skills

| Skill | When it triggers |
|-------|-----------------|
| `a11y-design` | Reviewing a Figma file or mockup before handoff — contrast, focus order, target size, screen-reader specs |
| `a11y-dev` | Writing or reviewing interactive components — semantic HTML, ARIA, keyboard nav, focus management |
| `a11y-qa` | Running an a11y audit before merge or release — axe-core scans, jsx-a11y lint, WCAG checks |

## Install

```
/plugin marketplace add Studio-Manfred/manfred-shared-knowledge
/plugin install manfred-a11y@manfred
```
