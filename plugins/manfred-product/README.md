# manfred-product

Product brief and PRD scaffolding using the Scandic 8-section template.

## Skills

| Skill | When it triggers |
|-------|-----------------|
| `brief-prd` | Writing a Product Brief / PRD / Scandic brief — opportunity, hypothesis, risks, scope, next step |

## Install

```
/plugin marketplace add Studio-Manfred/manfred-shared-knowledge
/plugin install manfred-product@manfred
```
