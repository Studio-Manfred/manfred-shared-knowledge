# manfred-knowledge

Knowledge-management utilities for personal vaults and document conversion.

## Skills

| Skill | When it triggers |
|-------|-----------------|
| `markitdown-convert` | Batch-converting PDFs, Word docs, PowerPoints, HTML, etc. to Markdown via the markitdown CLI |
| `clippings-linter` | Linting an Obsidian Clippings folder — broken notes, missing tags, duplicates, orphans |

## Install

```
/plugin marketplace add Studio-Manfred/manfred-shared-knowledge
/plugin install manfred-knowledge@manfred
```
